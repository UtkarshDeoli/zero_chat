# zero_chat

This is an Annanomous Chat platform 
We take ZERO data


## Getting Started

#### How to setup ?
1. Clone the repo <br>
    `git clone https://github.com/UtkarshDeoli/zero_chat.git`
        
2. cd to zero_chat
    `cd zero_chat`
        
3. Get Dependency
    `flutter pub get`

4. Create
    `flutter create .`

5. Run
    `flutter run`

- Or just copy this to your terminal:
```bash
  git clone https://github.com/UtkarshDeoli/zero_chat.git
  cd zero_chat
  flutter pub get
  flutter create .
  flutter run
```
